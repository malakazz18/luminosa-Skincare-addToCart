# luminosa-Skincare-addToCart
e-commerce skincare shop built with PHP/MySQL, featuring a dynamic shopping cart, order management, and responsive UI. Academic project for web development course.
 Features

- 🛍️ Product catalog with stock status badges
- 🛒 Real‑time shopping cart (add/update/remove items)
- 💾 Cart persistence via PHP sessions
- 📦 Order placement with stock reduction
- ✅ Order confirmation page
- 📱 Fully responsive design (mobile + desktop)




## Tech Stack

| Layer        | Technology                          |
|--------------|-------------------------------------|
| Backend      | PHP 8+ (native, no framework)       |
| Database     | MySQL                               |
| Frontend     | HTML5, CSS3, JavaScript (ES6)       |
| Cart Logic   | PHP Sessions + AJAX (fetch)         |
| Styling      | CSS custom properties + Flex/Grid   |
